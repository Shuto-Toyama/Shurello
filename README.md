# README

## Shullelo
 
タスク管理アプリ[trello](https://trello.com/)風のタスク管理アプリです。

## DEMO

https://shurello.herokuapp.com/

## 動作環境
  - ruby 2.7.2p137
  - Rails 6.0.3.5
  - Java Script
  - JQuery 
  - bootstrap
  - PostgreSQL 13.2
  - heroku/7.53.0

## 機能一覧
　　・ユーザー登録機能
　　・カード登録/削除/編集機能
 ・リスト登録/削除/編集機能

## 難しかった点
　　　・既存のToDoアプリの様な直感的に使用できるように構成（非同期通信など）
